## Giscus Comments
1. Create a public GitHub repository for comments.
2. Enable `Discussions` in that repository.
3. Install the giscus app on the repository.
4. Open [giscus.app](https://giscus.app/) and copy the repository and category values.
5. Fill in `.env.local` using `.env.example`.

This project uses:
- `data-mapping="pathname"`
- `data-theme="preferred_color_scheme"`
- `data-lang="ko"`
